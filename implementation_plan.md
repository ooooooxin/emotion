# 模拟捏史莱姆（Slime）解压小程序实现方案

开发一款集“多点触控流体形变物理反馈”与“实时合成 ASMR 白噪音沉浸”于一体的史莱姆解压捏捏乐功能，极大提升小程序的解压体验与用户停留时长。

## 用户审核项 (User Review Required)

> [!IMPORTANT]
> **1. 双 Canvas 2D 元球 (Metaball) 流体渲染方案**
> 为了在微信小程序性能受限的环境下实现极致流畅的史莱姆“果冻感”与“粘滞性”融合，我们将采用底层 Canvas 施加 CSS 滤镜 `filter: blur(12px) contrast(20)` 的元球融合技术。顶层 Canvas 保持无滤镜，用于渲染高清晰度的装饰配件（如金葱粉、小星星、水果切片等），保持精致度。
>
> **2. 基于 Web Audio API 的实时 ASMR 声音合成器**
> 无需下载任何音频文件，通过代码直接生成正弦波与白噪音，并根据用户的触摸速度与揉捏力度实时调节频率和音量。可完美模拟清脆的气泡爆裂声（Pop）、黏糊的吧唧声（Squish）和纤维拉扯声（Stretch）。
>
> **3. 互动 DIY、情绪释放和陈列柜玩法**
> 包含三套基底材质物理参数（水弹史莱姆、绵密起泡胶、厚实牛头胶），并支持双色无极调色粒子渐变融合，以及“情绪释放百分比”任务体系和“我的史莱姆陈列柜”存档功能。

## 待讨论与确认问题 (Open Questions)

> [!NOTE]
> 1. **ASMR 音效自动播放策略**：部分浏览器/平台要求用户必须有首交互（Click/Touch）之后才能播放 Web Audio 音频。我们将在页面首次触摸或点击时激活 AudioContext，确保声音顺畅播放。
> 2. **触觉震动反馈 (Taptic Engine)**：触觉震动使用 `uni.vibrateShort` 在手机真机上提供轻微反馈，在 H5 网页预览时将做降级处理（仅有声音与视觉特效，不报错）。

## 拟更改文件 (Proposed Changes)

### 史莱姆模拟器模块

#### [NEW] [index.vue](file:///d:/test/emotion/src/pages/slime/index.vue)
新建史莱姆模拟捏捏乐主页面。主要功能包含：
- **物理与渲染循环 (Physics-Render Loop)**：基于 requestAnimationFrame 或 Canvas 自身的 setInterval 刷新，维护 60 个左右的相互吸引/排斥粒子，支持全局向心力实现“外力消失后缓慢恢复原状”。
- **双 Canvas 叠加结构**：
  - 底层 Canvas（应用 CSS `blur(12px) contrast(20)`）：绘制彩色圆形粒子，自动黏连、形变、拉断。
  - 顶层 Canvas（无滤镜）：在粒子位置同步渲染星星、草莓切片等装饰配件，并绘制气泡破裂时的水花散开粒子动效。
- **ASMR 合成器 (ASMR Audio Synth)**：封装 Web Audio API 的 OscillatorNode、GainNode 以及 BiquadFilterNode，实时根据手指滑动速度与方向调制咕噜捏合声，在气泡破裂时触发尖锐频率衰减音效。
- **DIY 调节控制面板**：
  - 材质选择器：切换水弹（流动/清脆/蓝色）、起泡胶（体积变大/吧唧/粉白）、牛头胶（阻力大/低沉/深绿）。
  - 双色无极混合盘：粒子碰撞时自动插值颜色，揉捏可使红黄两色混为橙色。
  - 配件添加：撒入金葱粉、五角星亮片、水果切片，跟随粒子做运动。
- **情绪值与奖励**：揉捏积累进度，满 100% 后触发撒花庆祝并解锁稀有配件。
- **陈列柜存档**：支持在 LocalStorage 保存自定义史莱姆。
- **激励视频模拟**：精美的广告播放弹窗，闭环解锁高级配料。
- **送给好友**：序列化当前配方，通过分享链接传递。

### 手动功能验证
1. **多点触控与流体形变**：单指戳按排斥、多指抓取粒子拖拽，拉伸至一定距离自动断裂，松手后粒子缓慢向中心聚拢。
2. **ASMR 音效**：触摸并拖动史莱姆时持续发出吧唧声，移动越快声音越密集；点击气泡时发出清脆“啵”声。
3. **颜色混合**：选择两种不同的颜色，开始揉捏，观察粒子颜色逐渐融合成渐变混色。
4. **陈列柜与存档**：点击“存入陈列柜”，输入名字保存。切换到陈列柜 Tab，查看罐中史莱姆。点击罐子可取出重新揉捏。
5. **分享测试**：点击“生成海报”查看 Canvas 生成的配方卡片。
